# Release Notes / Отчет

Date: 2026-04-13

## English

This public repo now gives OpenIA a clear front door.

It explains the daily dev log pipeline, links to the reusable `daily-dev-log-skill`, and shows how the project pieces fit together without exposing private logs or machine-specific noise.

## Русский

Теперь у OpenIA есть понятная публичная витрина.

В ней объяснён дневник разработки, есть ссылка на переиспользуемый `daily-dev-log-skill`, и видно, как части проекта связаны между собой, без приватных логов и лишнего шума с машины.

## 中文

现在 OpenIA 有了清晰的公开展示页。

这里说明了每日开发日志流程，链接到了可复用的 `daily-dev-log-skill`，也展示了项目各部分如何连接在一起，同时不暴露私密日志和机器相关噪音。

## What is intentionally omitted / Что намеренно не включено

- Local router logs and raw diagnostics output.
- Machine-specific caches and temporary files.
- Personal working notes that do not help a new visitor understand the project.
